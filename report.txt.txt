# Data Storage Technologies Report

## 1. Introduction  
Data storage can be handled using different technologies, including XML, JSON, relational databases, and plain text files. 

## 2. Comparison  
- **Plain Text**: Simple but lacks structure.  
- **Databases**: Great for complex data but requires setup.  
- **JSON**: Compact but lacks validation.  
- **XML**: Self-descriptive and structured.  

## 3. Why XML Was Chosen  
- Good for hierarchical data.  
- Compatible with XPath queries.  

## 4. Conclusion  
XML is flexible for small datasets, while databases are better for large-scale applications.
